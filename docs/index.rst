.. |pt| replace:: pagexml-tools

|pt|
===============

Some tools for working with pagexml <https://www.primaresearch.org/tools/PAGELibraries> files.


Installation
------------

using pip:

.. code-block:: bash

    pip install pagexml-tools

using poetry:

.. code-block:: bash

    poetry add pagexml-tools

Read more on `basic usage <usage.html>`_
